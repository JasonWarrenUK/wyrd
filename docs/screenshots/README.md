# Screenshots

[Freeze](https://github.com/charmbracelet/freeze) captures terminal output as PNG or SVG images.

## Prerequisites

Install Freeze:

    go install github.com/charmbracelet/freeze@latest

## Usage

Regenerate all screenshots:

    make screenshots

## Convention

Screenshot scripts in this directory produce images in `docs/screenshots/output/`. Scripts are checked into version control; generated images are not.
